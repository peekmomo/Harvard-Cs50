🚀 My CS50x 2026 Journey | 程序员进化日记

欢迎来到我的计算机科学通关大本营！这里记录了我从零开始挑战哈佛大学 CS50x (Introduction to Computer Science) 的全过程。

我建立这个仓库的目的，是为了记录每天的听课心得、代码演练以及修 Bug 的血泪史。通过保持 GitHub 绿格子的更新，见证自己从小白到开发者的蜕变！

🗺️ 技能通关进度条 (Roadmap)

按部就班，绝不中途放弃！(完成一章就把 [ ] 改成 [x])

  • Week 0: Scratch —— 编程初体验，逻辑思维建立 🎮
  • Week 1: C —— 深入底层，向语法圣殿进发 💻
  • Week 2: Arrays —— 看看数据在内存里是怎么排排坐的 📊
  • Week 3: Algorithms —— 经典算法大作战（大 O 表示法、检索与排序）🕵️‍♂️
  • Week 4: Memory —— 传说中的“指针深水区”，死磕到底！🧠
  • Week 5: Data Structures —— 链表、树、哈希表，解锁高级内功 🌲
  • Week 6: Python —— 降维打击！体验现代语言的丝滑 🐍
  • Week 7: SQL —— 玩转数据库，数据增删改查的基本功 🗄️
  • Week 8: HTML, CSS, JavaScript —— 踏入前端大门，让代码长出漂亮的脸蛋 🌐
  • Week 9: Flask / Web —— 真正的全栈串联，前后端打通 ⚡
  • Week 10: CyberSecurity / Emoji —— 拓展视野 🔒
  • 🏆 Final Project —— 做出属于我自己的独立作品！🔥

📂 目录结构说明

为了方便日后查阅，本仓库的文件按照以下结构组织：

  • Week0-Scratch/：第 0 周的 Scratch 项目灵感与思维导图。
  • Week1-C/：C 语言基础笔记，以及各种练习写的小 demo（比如 hello.c, cash.c）。
  • Notes/：我自己整理的、官方笔记之外的个性化痛点总结。
