# CS50：二进制与十进制

## 一、什么是十进制？

我们平时使用的数字系统叫做**十进制（Decimal）**。

例如：

```text
365
```

实际上表示：

```text
3 × 100
+
6 × 10
+
5 × 1
```

也可以写成：

```text
3 × 10²
+
6 × 10¹
+
5 × 10⁰
```

### 十进制每一位的权重

| 位数 | 权重         |
| -- | ---------- |
| 个位 | 10⁰ = 1    |
| 十位 | 10¹ = 10   |
| 百位 | 10² = 100  |
| 千位 | 10³ = 1000 |

规律：

```text
每往左一位 ×10
```

因为十进制是：

```text
逢10进1
```

---

## 二、什么是二进制？

二进制（Binary）只允许使用两个数字：

```text
0
1
```

因为只有两个数字，所以：

```text
逢2进1
```

例如：

```text
0
1
```

再加 1：

```text
10
```

注意：

```text
10（二进制）
≠ 10（十进制）
```

它实际上表示：

```text
2（十进制）
```

---

## 三、二进制每一位代表什么？

十进制：

```text
1000 100 10 1
```

对应：

```text
10³ 10² 10¹ 10⁰
```

---

二进制：

```text
8 4 2 1
```

对应：

```text
2³ 2² 2¹ 2⁰
```

### 二进制权重表

| 位数  | 权重       |
| --- | -------- |
| 第1位 | 2⁰ = 1   |
| 第2位 | 2¹ = 2   |
| 第3位 | 2² = 4   |
| 第4位 | 2³ = 8   |
| 第5位 | 2⁴ = 16  |
| 第6位 | 2⁵ = 32  |
| 第7位 | 2⁶ = 64  |
| 第8位 | 2⁷ = 128 |

规律：

```text
每往左一位 ×2
```

---

## 四、为什么是 1、2、4、8？

### 1位二进制

可表示：

```text
0
1
```

共 2 个数字。

---

### 2位二进制

可表示：

```text
00
01
10
11
```

对应：

```text
0
1
2
3
```

共 4 个数字。

---

### 3位二进制

可表示：

```text
000
001
010
011
100
101
110
111
```

对应：

```text
0 ~ 7
```

共 8 个数字。

---

因此：

```text
第1位 = 1
第2位 = 2
第3位 = 4
第4位 = 8
```

这是为了能够表示越来越大的数字。

---

## 五、二进制转十进制

原则：

> 哪一位是 1，就把对应权重加起来。

---

### 示例1

```text
1011
```

对应权重：

```text
8 4 2 1
```

计算：

```text
8 + 0 + 2 + 1
```

结果：

```text
11
```

所以：

```text
1011₂ = 11₁₀
```

---

### 示例2

```text
1101
```

对应权重：

```text
8 4 2 1
```

计算：

```text
8 + 4 + 0 + 1
```

结果：

```text
13
```

所以：

```text
1101₂ = 13₁₀
```

---

## 六、十进制转二进制

以数字 13 为例。

权重：

```text
8 4 2 1
```

### 第一步

13 能放 8

```text
13 - 8 = 5
```

记录：

```text
1
```

---

### 第二步

5 能放 4

```text
5 - 4 = 1
```

记录：

```text
1
```

---

### 第三步

1 不能放 2

记录：

```text
0
```

---

### 第四步

1 能放 1

```text
1 - 1 = 0
```

记录：

```text
1
```

得到：

```text
1101
```

所以：

```text
13₁₀ = 1101₂
```

---

## 七、CS50 的灯泡模型

CS50 经常用灯泡来表示二进制。

```text
128 64 32 16 8 4 2 1
 □   □  □  □ □ □ □ □
```

规则：

```text
亮 = 1
灭 = 0
```

例如：

```text
0 0 0 0 1 1 0 1
```

对应：

```text
8 + 4 + 1
```

结果：

```text
13
```

---

## 八、位（Bit）和字节（Byte）

### Bit

一个二进制位：

```text
0
或
1
```

称为：

```text
1 bit
```

---

### Byte

8 个 bit：

```text
00000000
```

称为：

```text
1 byte
```

一个 Byte 可以表示：

```text
00000000 ~ 11111111
```

即：

```text
0 ~ 255
```

共：

```text
256 个数字
```

---

## 九、核心总结

### 十进制

```text
个位   = 10⁰
十位   = 10¹
百位   = 10²
千位   = 10³
```

规律：

```text
每往左 ×10
```

---

### 二进制

```text
1
2
4
8
16
32
64
128
```

规律：

```text
每往左 ×2
```

---

### 二进制转十进制

```text
把所有为 1 的位对应权重加起来
```

例如：

```text
1011
=
8 + 2 + 1
=
11
```

---

### 十进制转二进制

```text
从最大的 2 的幂开始减
能放记 1
不能放记 0
```

例如：

```text
13
=
8 + 4 + 1
=
1101
```

---

## CS50 最重要的一句话

十进制和二进制本质上是同一种思想。

区别仅仅在于：

```text
十进制：每往左 ×10
二进制：每往左 ×2
```

理解这一点后，后续的：

* Bit
* Byte
* ASCII
* Unicode
* 图片存储
* 音频存储
* 视频存储

都会建立在同一个基础之上。
